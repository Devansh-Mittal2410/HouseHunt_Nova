## 📌 About The Project

**HouseHunt Nova** is a full-stack real estate web application built using the **MERN Stack (MongoDB, Express, React, Node.js)**.

It allows users to:

- Browse property listings  
- View detailed property information  
- Explore homes through a clean, responsive UI  
- Interact with a RESTful backend connected to MongoDB  

This project demonstrates real-world full-stack development, API integration, database modeling, and deployment.
